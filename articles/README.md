# 绿链汇公众号文章管理

## 文件说明

- `data.js` - 文章数据文件，包含所有公众号文章信息

## 如何添加新文章

1. 打开 `data.js` 文件
2. 在 `ARTICLES` 数组的**最前面**添加新文章（最新的文章放最前面）
3. 按以下格式填写：

```javascript
{
  title: "文章标题",
  url: "https://mp.weixin.qq.com/s/xxxxxx",
  digest: "文章摘要（简短描述文章内容）",
  date: "2025-03-20",
  color: "#00A86B"  // 渐变色背景颜色
}
```

## 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| title | 文章标题 | "绿链未来.聚力共生" |
| url | 公众号文章链接 | "https://mp.weixin.qq.com/s/xxxxxx" |
| digest | 文章摘要（30-50字） | "绿链汇发起成立..." |
| date | 发布日期 | "2025-03-18" |
| color | 封面背景色 | "#00A86B" |

## 颜色推荐

```javascript
"#00A86B"  // 主绿色
"#2E8B57"  // 海绿
"#3CB371"  // 中海绿
"#228B22"  // 森林绿
"#32CD32"  // 酸橙绿
"#00CED1"  // 深青色
"#20B2AA"  // 浅海绿
"#5F9EA0"  // 军绿
"#6B8E23"  // 橄榄绿
"#556B2F"  // 深橄榄绿
```

## 注意事项

1. **文章顺序**：最新的文章放在数组最前面
2. **日期格式**：统一使用 `YYYY-MM-DD` 格式
3. **摘要长度**：建议30-50字，简明扼要
4. **颜色选择**：选择绿色系颜色，保持视觉统一

## 关于图片

由于微信公众号图片有防盗链限制，外部网站无法直接显示公众号图片。
因此采用**渐变色背景+标题文字**的方式作为封面显示。

如需使用真实图片，请：
1. 下载文章第一张图片到 `images/` 文件夹
2. 修改 `data.js` 中的 `image` 字段为图片路径
